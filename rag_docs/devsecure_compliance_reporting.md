
# DevSecure Compliance and Reporting

## Security reporting
DevSecure provides reports for:
- vulnerability tracking
- remediation progress
- dependency risk
- secret exposure

## Compliance frameworks
Reports can support compliance with standards such as:
- SOC2
- ISO 27001
- NIST secure development practices

## Exporting reports
Reports can be exported from the DevSecure dashboard for audits or compliance reviews.

## SBOM reporting
DevSecure can generate Software Bill of Materials reports listing all application dependencies.

## Security analytics
Dashboards provide visibility into:
- vulnerability trends
- remediation timelines
- repository security posture
