
# DevSecure Admin Configuration Guide

## Workspace administration
Workspace administrators manage:
- repositories
- users and permissions
- integrations
- security policies

## User roles
Typical roles include:
- Administrator
- Security Engineer
- Developer

## Adding team members
Administrators can invite members from workspace settings and assign roles.

## Configuring scan policies
Policies allow teams to:
- block critical vulnerabilities
- enforce secret detection
- require dependency updates

## Managing repository integrations
Repositories can be connected or removed through the integrations dashboard.
